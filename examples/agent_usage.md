# Agent Usage Examples

## Check balances (read-only)
```text
Use $grid-wizard to run `health_check` and `show_config`, then summarize the safe runtime state without placing orders.
```

## Validate environment
```text
Use $grid-wizard to run `validate_env` and explain what is missing for dry-run and live mode.
```

## Simulate one cycle
```text
Use $grid-wizard to run `simulate_cycle` with current config and summarize what would happen in dry-run.
```

## Preview dry-run placement
```text
Use $grid-wizard to run `dry_run_place_preview` and summarize the preview output without executing anything.
```

## Ask about experimental live mode
```text
Explain that live actions are experimental and not part of the public v1 release. Tell me what would need to be validated before they could be considered.
```
